# projeto-pagina-tonimcell

Projeto desenvolvido para a loja de celulares do meu pai, a Tonim Cell, com o objetivo de criar uma presença digital moderna, profissional e responsiva para divulgação dos serviços e produtos da loja.

A página foi desenvolvida com foco em experiência do usuário, navegação intuitiva e design moderno, utilizando animações, efeitos visuais e uma interface adaptável para celulares, tablets e computadores.

O site possui integração direta com os canais de atendimento da loja, onde os botões de WhatsApp e mensagem direcionam os clientes para o WhatsApp oficial da Tonim Cell, facilitando o contato rápido para orçamentos, suporte e atendimento. Os botões de Instagram levam diretamente para o perfil oficial da loja, aumentando a divulgação nas redes sociais e a interação com os clientes.

Também foi implementada uma seção interativa de localização com design de mapa, que redireciona os usuários para a localização da loja no Google Maps, facilitando o acesso presencial dos clientes.

Tecnologias utilizadas no projeto:
• HTML5 — estruturação da página
• CSS3 — estilização, responsividade, animações e efeitos visuais
• JavaScript — interatividade e funcionalidades dinâmicas
• Git e GitHub — versionamento e hospedagem do projeto
• GitHub Pages — publicação do site online

Este projeto foi importante para aprimorar minhas habilidades em desenvolvimento front-end, responsividade, organização de layout e integração de links externos para comunicação e localização comercial.
